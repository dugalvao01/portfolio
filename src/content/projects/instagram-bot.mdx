## Visão Geral

Desenvolvimento de um bot para automatizar tarefas repetitivas de marketing social, respeitando os limites da API para evitar bloqueios.

## Tecnologias

- **Python**: Linguagem core.
- **Selenium**: Para emulação de interações reais no browser.
- **Pandas**: Para gerenciar base de dados de leads.

## Arquitetura

O script roda em um container Docker agendado via Cron, garantindo execução diária sem intervenção manual.

```python
def interagir_com_hashtags(tag_list):
```

Economia de 15 horas semanais da equipe de marketing.
